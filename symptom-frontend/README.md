🛠 The Stack
UI: React 18, Tailwind CSS, Framer Motion (animations), Lucide (icons).

State: Context API for global Auth and react-hot-toast for real-time feedback.

Routing: React Router v6 with a centralized AppLayout wrapper.





 Highlights
src/services/api.js: All backend communication logic in one place.

src/pages/SymptomCheckerPage.jsx: The core AI analysis interface.

src/index.css: Custom CSS design system (colors, fonts, glassmorphism).
